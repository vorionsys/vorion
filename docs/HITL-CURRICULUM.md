## Foundation Track (H1 Observer)

**Public Title**: "Introduction to AI Oversight"
**Future Badge**: H1 Observer
**Time Estimate**: 4-6 hours total

### Course 101: AI Agent Fundamentals
**Duration**: 1 hour
**Learning Objectives**:
- Understand what an AI agent is vs traditional software
- Recognize agent autonomy levels
- Identify when an agent is making decisions

**Modules**:
1. What is an AI Agent? (15 min)
2. Agents vs Automation vs Scripts (15 min)
3. Autonomy Spectrum: Tool → Assistant → Agent → Autonomous (15 min)
4. Quiz & Reflection (15 min)

### Course 102: Trust Tiers Explained
**Duration**: 1.5 hours
**Learning Objectives**:
- Name all 8 trust tiers (T0-T7)
- Understand what each tier allows an agent to do
- Recognize why agents start at T0

**Modules**:
1. The Trust Ladder: T0 to T7 (20 min)
2. What Each Tier Permits (30 min)
3. How Agents Earn Trust (20 min)
4. How Agents Lose Trust (20 min)
5. Quiz & Scenarios (20 min)

### Course 103: Recognizing Agent Failures
**Duration**: 1.5 hours
**Learning Objectives**:
- Identify common agent failure modes
- Distinguish between errors and misbehavior
- Know when to intervene vs when to let agent self-correct

**Modules**:
1. Failure Types: Errors, Hallucinations, Misbehavior (25 min)
2. Red Flags: What to Watch For (25 min)
3. Intervention Decision Tree (25 min)
4. Case Studies: Good vs Bad Interventions (25 min)

### Course 104: Basic Oversight Practices
**Duration**: 1 hour
**Learning Objectives**:
- Apply the "trust but verify" principle
- Use basic monitoring tools
- Document observations properly

**Modules**:
1. Trust But Verify: Principles (20 min)
2. Monitoring Dashboard Walkthrough (20 min)
3. Writing Good Observations (20 min)

**Assessment**: Foundation Track Quiz (must score 80%+)

## Tracking & Future Badge Credit

### Completion Tracking Schema

```typescript
interface CourseCompletion {
  userId: string;
  courseId: string;           // e.g., "101", "301"
  completedAt: Date;
  score: number;              // Quiz/exam score
  timeSpentMinutes: number;
  platform: 'learn.vorion.org';
}

interface TrackCompletion {
  userId: string;
  track: 'foundation' | 'operator' | 'supervisor' | 'auditor' | 'architect';
  completedAt: Date;
  averageScore: number;

  // Future badge fields (null until Phase 2)
  badgeId: string | null;
  certificationLevel: 'H1' | 'H2' | 'H3' | 'H4' | 'H5' | null;
}
```

### Badge Migration (When Phase 2 Goes Live)

```sql
-- When we flip to Phase 2, auto-issue badges
INSERT INTO hitl_certifications (user_id, level, certified_at)
SELECT
  user_id,
  CASE track
    WHEN 'foundation' THEN 'H1'
    WHEN 'operator' THEN 'H2'
    WHEN 'supervisor' THEN 'H3'
    WHEN 'auditor' THEN 'H4'
    WHEN 'architect' THEN 'H5'
  END,
  completed_at
FROM track_completions
WHERE completed_at IS NOT NULL;
```
