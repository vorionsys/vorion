# Domain & Brand Alignment

## Organization Structure

### Vorion (Open Source - vorion.org)

Vorion is the open-source foundation providing standards and frameworks for AI agent governance.

| Project | Domain | Description |
|---------|--------|-------------|
| Vorion Main | vorion.org | Main organization site |
| BASIS | basis.vorion.org | Baseline Authority for Safe & Interoperable Systems |
| CAR ID | carid.vorion.org | Categorical Agentic Registry specification |
| ATSF | atsf.vorion.org | Agent Trust & Safety Framework |
| Learn | learn.vorion.org | Documentation, tutorials, and educational resources |

### Agent Anchor AI (Private Company - agentanchorai.com)

Agent Anchor AI is the commercial entity building enterprise products on Vorion standards.

| Product | Domain | Description |
|---------|--------|-------------|
| Agent Anchor Trust | trust.agentanchorai.com | Trust verification and certification |
| Agent Anchor Logic | logic.agentanchorai.com | Policy engine and governance logic |
| Cognigate | cognigate.dev | AI gateway and routing service |
| API | api.agentanchorai.com | Production API endpoint |
| Status | status.agentanchorai.com | Service status page |
